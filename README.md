# Romos Junction

Authentic North Indian Vegetarian Cuisine — Sikandra, Agra

## Project Info

A showcase website for **Romos Junction**, a popular vegetarian restaurant in Sikandra, Agra known for its authentic North Indian flavors.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

## Deployment

Build the project for production:

```sh
npm run build
```
